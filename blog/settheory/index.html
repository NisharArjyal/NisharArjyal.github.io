Preface:

I have been learning axiomatic set theory lately and would like to use this blog as a way to document the notes as I learn them. If you are not a student of mathematics who somehow stumbled upon this post for learning just the basic amount of set theory for learning other subjects in applied mathematics then you shouldn't fret. Basic Definitions:

We will separate the notion of classes and sets. All objects are classes and those classes that can be contained in other classes will be referred to as elements or sets. We will consider the ∈\in∈ to be a binary, the membership relation along with the === the equality predicate. We are now ready to make our first definition that will use formal notation.

Definition:

A=B: Two classes AAA and BBB are said to be equal if every class that has AAA as an element also has the class BBB as an element and vice versa.

(A=B) ⟺ (∀X)[A∈X ⟺ B∈X](A = B) \iff (\forall X)[A\in X \iff B\in X] (A=B)⟺(∀X)[A∈X⟺B∈X]

We will also make the following definition for subsets.

Definition:

A⊆BA\subseteq BA⊆B: AAA is said to be a subset of BBB if every element of AAA is also an element of BBB.

A⊆B ⟺ ∀t[t∈A ⟹ t∈B]A\subseteq B \iff \forall t[t\in A \implies t \in B] A⊆B⟺∀t[t∈A⟹t∈B]

ZFC Axioms:

So, we will begin by mentioning the axioms of Zermelo and Fraenkel as mentioned in common textbooks on set theory like Jech (2003), Enderton (1977).

Axioms:

  1. Axiom of Extensionality: If AAA and BBB have the same elements, then they are equal.

∀A∀B[∀x(x∈A ⟺ x∈B) ⟹ A=B] \forall A \forall B [\forall x (x\in A \iff x \in B) \implies A = B] ∀A∀B[∀x(x∈A⟺x∈B)⟹A=B]

or alternatively

∀A∀B[∀x(x∈A ⟺ x∈B) ⟹ ∀X(A∈X ⟺ B∈X)] \forall A \forall B [\forall x (x\in A \iff x\in B) \implies \forall X (A\in X \iff B \in X)] ∀A∀B[∀x(x∈A⟺x∈B)⟹∀X(A∈X⟺B∈X)]

  1. Axiom of Empty Set: There exists a set having no elements.

∃A(∀x,x∉A) \exists A (\forall x, x\notin A) ∃A(∀x,x∈/A)

  1. Axiom of Pairing: For any AAA and BBB, there exists a set that contains exactly AAA and BBB.

∀A∀B∃C[∀x(x∈C ⟺ x=A∨x=B)] \forall A \forall B \exists C [\forall x (x \in C \iff x = A \lor x = B)] ∀A∀B∃C[∀x(x∈C⟺x=A∨x=B)]

  1. Axiom of Power Set: For any set A, there exists a set whose elements are exactly the subsets of AAA.

∀A∃B∀x(x⊆A ⟺ x∈B) \forall A \exists B \forall x (x\subseteq A \iff x\in B) ∀A∃B∀x(x⊆A⟺x∈B)

  1. Axiom Schema of Separation: Let φ\varphiφ be any formula with all free variables among x,c,t1,…,tnx, c, t_{1}, \dots, t_{n}x,c,t1​,…,tn​ (BBB is not free in φ\varphiφ). Then,

∀t1∀t2…∀tn∀c∃B∀x[x∈B ⟺ (x∈c)∧φ(x,c,t1,…,tn)] \forall t_{1}\forall t_{2}\dots\forall t_{n}\forall c \exists B \forall x[x\in B \iff (x\in c) \land \varphi(x,c, t_{1}, \dots, t_{n})] ∀t1​∀t2​…∀tn​∀c∃B∀x[x∈B⟺(x∈c)∧φ(x,c,t1​,…,tn​)]

  1. Axiom of Union: For any AAA, there exists a set whose elements are exactly the members of members of AAA.

∀A∃B∀x[x∈B ⟺ (∃a∈A)x∈a] \forall A \exists B \forall x [x\in B \iff (\exists a\in A) x\in a] ∀A∃B∀x[x∈B⟺(∃a∈A)x∈a]

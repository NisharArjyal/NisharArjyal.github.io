Preface

Basically, the people in the maths discord have announced a measure theory reading group that meets every Saturdays. I have been wanting to learn measure theory for a while and I see this as a perfect opportunity to finally start learning it given that it is enormously helpful having other people studying it together with you trying to cover a chapter each week. So, consider this blog as a place for my notes as I study measure theory. As a reference, the group will primarily be using Folland (1999) and so will I in these notes. If I use any other textbook, I will make sure to reference it later in this blog.

Introduction

We want a mapping μ:P(Rn)→[0,∞]\mu: \mathscr{P}(\mathbb R^{n}) \to [0,\infty]μ:P(Rn)→[0,∞] that takes subsets of Rn\mathbb R^{n}Rn, n∈N\{0}n\in \mathbb N\backslash \{0\}n∈N\{0} to a non-negative real number (extended to +∞+\infty+∞). It turns out that the three properties P-1, P-2 and P-3 are mutually inconsistent.

Theorem:
There is no such map μ:P(R)→[0,∞]\mu: \mathscr{P}(\mathbb R) \to [0,\infty]μ:P(R)→[0,∞] that satisfies all P-1, P-2 and P-3
Proof:

Define a relation ∼\sim∼ on [0,1)[0,1)[0,1) as x∼y ⟺ x−y∈Qx\sim y \iff x-y\in \mathbb Qx∼y⟺x−y∈Q. Clearly, ∼\sim∼ is an equivalence relation. Consider Λ=[0,1) ⁣/ ⁣∼\Lambda = % [0,1)\!/\!{\sim}%Λ=[0,1)/∼ the set of all equivalence classes of [0,1)[0,1)[0,1) under the equivalence relation ∼\sim∼.

Let us invoke the Axiom of Choice on Λ\LambdaΛ to pick from each equivalence class in Λ\LambdaΛ precisely one member and call this set NNN such that N⊆[0,1)N\subseteq [0,1)N⊆[0,1).

Now, let R=Q∩[0,1)R = \mathbb Q \cap [0,1)R=Q∩[0,1). For each r∈Rr \in Rr∈R, let

Nr={x+r∣x∈N∩[0,1−r)}∪{x+r−1∣x∈N∩[1−r,1)}.N_{r} = \{x+r\mid x\in N \cap [0,1-r)\} \cup \{x+r-1 \mid x\in N\cap [1-r, 1)\}.Nr​={x+r∣x∈N∩[0,1−r)}∪{x+r−1∣x∈N∩[1−r,1)}.
